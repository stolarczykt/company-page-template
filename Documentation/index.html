Introduction

BeeEater is a fully responsive web template. It comes with 8 predefined color themes and bonus blog and post pages. Developer version uses LESS (or SASS) so it is easy to modify and you are able to create as many color themes as you want.

Features

  • Fully responsive design
  • HTML5 and CSS3
  • Blog and post pages
  • Build with LESS and SASS
  • 8 predefined color themes - you can build new one by changing few variables
  • Integrations with Font Awesome - Are you bored with circles? Replace them with squares or bells... Want more? - Draw custom skills charts.
  • Skills charts - customizable by Font Awesome icons
  • Isotope filterable portfolio
  • Bootstrap's full screen Carousel
  • Realtime form validation
  • Developer version - Grunt tasks like watch, livereload, less or sass.
  • Documentation

Installation

This template consists of static HTML files, so you are able to view them via web browser on your desktop.
If you have some knowledge about Grunt, you can run server locally by calling: 

$ grunt server

Server by default will be listening on localhost:9000. More about developer version will be covered in the next section.

You can install template on your server by uploading content of Template directory into your server's root directory. Then go to your domain to check if everything works.

Developer's version

If want to use developer's version, you will need to install NPM on your local machine. NPM comes along with Node.js.

To be more effective at working with developer's version you need to have some basic knowledge about Grunt.

Developer's version was mainly created to give you opportunity to build infinity number of color themes, by changing only a few variables. Moreover beacause of watch task it's easy to have realtime experience during development.

Developer's version is located under Template-Dev directory. Before you start to play with it, you need to install all dependencies. It can be done by calling: 

$ npm install

After that you are ready to use Grunt and it's tasks which are defined in Gruntfile.js.

Grunt tasks

Some of Grunt tasks which are available for you:

  • less:compile - will compile all themes (LESS files) which are located under css-langs/less/colors/ directory, and will place them under resources/css/ directory, overriding the old ones.
  • sass:compile - will compile all themes (SASS files) which are located under css-langs/sass/colors/ directory, and will place them under resources/css/ directory, overriding the old ones.
  • watch - will watch changes on files and then trigger livereload.
  • clean:dist - will clean dist directory.
  • copy:dist - will copy template files to dist directory.
  • connect:dev - will start a local static web server on 9000 port. Moreover adds livereload script to the response.
  • server - starts a local static web server on 9000 port and reloads content each time files are changed.
  • build - builds template: compiles LESS or SASS files, cleans dist directory and copies a new version of template to this directory.

Creating new color themes

Creating new color theme can be done by following these steps:

  1. Go to css-langs/less/colors/ directory and create new LESS file (e.g. new-theme.less) with identical content as other files in this directory.
  2. Change color variables in newly created file.
  3. Run less:compile task.
  4. New theme file (new-theme.css) will be created in resources/css/ directory.

Skills charts

Drawing skills charts is done by great jquery-plugin-circliful.
Big advantage of using this plugin is integration with Font Awesome icons.

Most important data-* attributes for you, are:

  1. data-text - big text displayed inside of the circle
  2. data-percent - chart's filling percentage
  3. data-icon - Font Awesome icon which you want to use inside of the circle
  4. data-info - smaller text displayed inside of the circle

Notice: you can change color themes on your website, so we decided to set color of the chart dynamically. That means that we are setting this color by JavaScript (main.js) and by default it uses styles from footer element. Because of that it is important to do not delete this element.

If you want to add new chart you should:

  1. Create new chart box with new ID add all needed data:
  2. Add ne line in main.js file for new created chart: 
    									//drawing skills charts
									var el=document.getElementById('footer');
									var cs=document.defaultView.getComputedStyle(el,null);
									var bg=cs.getPropertyValue('background-color');

									$('#skillNo1').html("").data("fgcolor", bg).data("iconcolor", bg).circliful();
									...
									...
									$('#skillNo5').html("").data("fgcolor", bg).data("iconcolor", bg).circliful();
									//EOF skills charts

For more detailed documentation you should go to https://github.com/pguso/jquery-plugin-circliful.

Google Maps

Using Google Maps is very simple. All you need to do is go to main.js and edit coordinates in LatLng() function. 

							function initGoogleMap() {
								var var_location = new google.maps.LatLng(45.430817,12.331516); //change coordinates here

								var var_mapoptions = {
									scrollwheel: false,
									center: var_location,
									zoom: 14
								};

								var var_marker = new google.maps.Marker({
									position: var_location,
									map: var_map,
									title:"Venice"}); //change marker's title

								var var_map = new google.maps.Map(document.getElementById("map-container"),
									var_mapoptions);

								var_marker.setMap(var_map);
							}
